## 4. 给 ResearchOS 的建议

更稳的论文链路通常是：

1. `PaperQA2 / paper-glance-skill` 负责读和证据抽取
2. `OpenDraft / Anthropic Skills` 负责组织初稿
3. `Open Research` 负责 critique 与 reviewer response
4. `Auto-Resubmit + Tectonic / latexmk` 负责投稿末端

如果你的目标是“辅助研究”，这条链路通常比让一个模型直接从零生成全文更靠谱。
