# 选题用 Skills、MCP 与 Agent 详表

这一页把“选题阶段”最常用的能力层拆成三组：`Skills`、`MCP` 和 `Agent / 工作台`。适合在已有模糊兴趣、但还没有形成稳定研究问题时使用。

## 四、常见使用顺序

1. 先用 `Paper Search MCP / OpenReview MCP / OpenAlex MCP` 建立候选论文池。
2. 再用 `Connected Papers / ResearchRabbit / Litmaps` 做网络扩展。
3. 对关键论文使用 `paper-glance-skill / PaperQA2 / research-lookup` 做精读。
4. 用 `Open Research / RTI / GPT Researcher / STORM` 形成 topic dossier。
5. 再进入 [04-新颖性判断与综述](../04-新颖性判断与综述/README.md) 做 novelty 和 reviewer-style critique。
