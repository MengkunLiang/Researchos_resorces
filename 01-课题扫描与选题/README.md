# 课题扫描与选题

这一模块聚焦方向扫描、前沿追踪、候选问题形成、研究切口判断与早期可行性评估。核心目标不是“直接定题”，而是把模糊兴趣逐步收敛成可验证、可复现、可投稿的研究问题。

## 这一步通常要回答的问题

1. 当前方向最近 6 到 18 个月在讨论什么。
2. 哪些问题已经很拥挤，哪些问题还有明显空白。
3. 哪些会议、期刊、实验范式和数据资源最值得优先关注。
4. 一个候选想法究竟是“有意思”，还是“足够新、足够可做、足够能讲清楚”。

## 六、推荐组合

1. `paper-search-mcp + openreview-mcp + openalex-mcp`
   适合做 broad search、会议追踪和主题 landscape。
2. `Connected Papers / ResearchRabbit / Litmaps + OpenAlex`
   适合从 seed paper 扩展到研究家族和引文网络。
3. `Open Research / RTI / GPT Researcher / STORM`
   适合把主题扫描快速升级成可读报告和问题树。
4. `PaperQA2 + paper-glance-skill + scite`
   适合把候选方向里的关键论文做精读、证据核验和支持/质疑分析。
