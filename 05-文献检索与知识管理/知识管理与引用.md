# 知识管理与引用

科研工作流真正可持续的部分，通常不是一次性的对话结果，而是可复用、可追溯、可迁移的文献库、批注、Bib、笔记与写作资产。

## 5. 可持续的资产分层

### 层 1：原始资产

- 论文 PDF、补充材料、网页快照。
- DOI、arXiv ID、PMID、OpenReview URL、出版社链接。
- BibTeX / BibLaTeX / RIS 等可导出元数据。

### 层 2：结构化资产

- 论文笔记、批注、高亮、摘录。
- claim list、comparison table、method taxonomy、baseline 清单。
- 主题标签、作者 / 机构索引、阅读状态与证据出处。

### 层 3：项目级资产

- 研究问题、假设、实验计划、复现实验记录。
- 写作提纲、草稿、投稿版本、审稿回复材料。
- 与代码仓库、数据版本、评测结果对应的交叉引用。

## 6. 实践原则

- 优先开放格式与本地可导出的资产，避免锁定在单一封闭平台。
- 引用键、Bib 导出与样式标准化越稳定，后续迁移成本越低。
- 文献管理器、笔记系统与写作工具的边界越清晰，长期维护越稳。
- 对关键资料保留原始链接与唯一标识符，有助于复查、复现与团队协作。
