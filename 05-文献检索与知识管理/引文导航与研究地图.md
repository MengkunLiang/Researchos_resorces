# 引文导航与研究地图

这一页聚焦引文追踪、文献可视化、研究地图与文献计量分析相关公开资源。

## 1. 可视化引文发现与导航

| 资源 | 类型 | 简述 | 链接 |
| --- | --- | --- | --- |
| ResearchRabbit | 文献发现平台 | 适合围绕 seed papers 扩展相关文献、作者网络与时间演化。 | [ResearchRabbit](https://www.researchrabbit.ai/) |
| Connected Papers | 研究地图平台 | 适合从单篇论文出发快速看到相近工作、prior works 与 derivative works。 | [Connected Papers](https://www.connectedpapers.com/) |
| Litmaps | 文献地图平台 | 适合做持续更新的 literature map、追踪新论文与协作整理。 | [Litmaps](https://www.litmaps.com/) |
| Inciteful | 引文网络工具 | 提供 paper discovery 与 literature connector，适合桥接两个研究簇。 | [Inciteful Docs](https://incitefulmed.com/academic/help/) |
| Open Knowledge Maps | 开放知识地图 | 以可视化方式展示主题下的研究簇，适合快速熟悉陌生领域。 | [Open Knowledge Maps](https://openknowledgemaps.org/) |
| scite | 引文上下文平台 | 强调 supporting / contrasting / mentioning 等 citation context。 | [scite](https://scite.ai/) |

| 资源 | 类型 | 简述 | 链接 |
| --- | --- | --- | --- |
| VOSviewer | 文献计量软件 | 适合构建引文、共词、共被引、作者合作与主题聚类图。 | [VOSviewer](https://www.vosviewer.com/) |

| Headstart | 开源知识地图框架 | Open Knowledge Maps 背后的开源可视化框架，适合自建研究地图。 | [OpenKnowledgeMaps/Headstart](https://github.com/OpenKnowledgeMaps/Headstart) |

## 3. 适用场景

| 场景 | 更适合先看什么 |
| --- | --- |
| 从一篇代表论文向外扩展 | `Connected Papers + ResearchRabbit` |
| 持续维护一个方向的文献版图 | `Litmaps + ResearchRabbit` |
| 找两个主题之间的桥接论文 | `Inciteful` |
| 观察引文是支持还是对立 | `scite` |
| 做系统的文献计量与主题演化图 | `VOSviewer + bibliometrix / Biblioshiny` |
| 希望自建开放研究地图 | `Open Knowledge Maps + Headstart` |

## 4. 使用提示

- 研究地图工具大多强调“关系发现”，不等同于系统综述意义上的完整检索。
- 引文图擅长帮助发现经典文献与研究簇，但对最新未形成稳定引用的工作不一定敏感。
- 做正式综述、benchmark 盘点或 novelty 分析时，通常仍需回到原始数据库与全文核查。
