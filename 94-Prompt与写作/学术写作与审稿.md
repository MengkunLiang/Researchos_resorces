## 4. 一条较稳的写作与审稿链路

| 阶段 | 适合优先参考的资源 |
| --- | --- |
| 证据抽取与笔记 | `PaperQA2`、`paper-glance-skill`、`GPT Academic` |
| 初稿与提纲组织 | `OpenDraft`、`ScholarCopilot`、`Paperpal`、`Jenni` |
| critique 与 rebuttal | `Open Research`、`AI Peer Review`、`OpenReview` |
| 排版、协作与投稿 | `Manubot`、`Quarto`、`Typst`、`Auto-Resubmit`、`Tectonic / latexmk` |

## 5. 写作类 AI 资源最适合承担什么角色

1. 帮助形成提纲、段落骨架、候选表达与 response draft。
2. 帮助把已有 evidence cards、paper cards、实验日志转成更清晰的文本。
3. 帮助在投稿前做语言、结构、引用与 rebuttal 轮次整理。

不太适合直接承担的部分：

1. 独立发明未经核验的研究结论。
2. 在没有证据链时自动生成引用密集段落。
3. 跳过实验、数据和审稿反馈，直接产出“最终可投版本”。
