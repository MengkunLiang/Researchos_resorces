# 系统综述与证据综合

这一页聚焦 literature review、rapid review、systematic review、research report 与证据综合写作。相比一般学术写作工具，这一类资源更强调检索、筛选、数据抽取、报告生成与可回查证据。

## 1. 研究报告与系统综述平台

| 资源 | 类型 | 简述 | 链接 |
| --- | --- | --- | --- |
| Elicit | 科学研究平台 | 面向 search、reports、systematic reviews、library 与 alerts 的一体化研究平台。 | [Elicit](https://elicit.com/) |
| Elicit Systematic Reviews | 系统综述工作流 | 官方系统综述入口，覆盖 gather、screen、extract、report 等步骤。 | [Elicit Docs](https://support.elicit.com/en/articles/7927169) |
| Elicit Report Templates | 报告模板 | 官方支持文章，强调 reports 与模板化输出能力。 | [Elicit Support](https://support.elicit.com/en/articles/10639553) |
| Consensus | 科学论文搜索引擎 | 强调科学论文搜索、AI synthesis 与 grounded answers。 | [Consensus](https://consensus.app/home/about-us/) |
| Consensus Pro Analysis | 研究摘要与 deliverables | 在文献发现基础上生成 outline、technical report、rapid review 等研究交付物。 | [Consensus Blog](https://consensus.app/home/blog/introducing-the-consensus-co-pilot/) |
| ScholarCopilot | citation-grounded academic writing | 兼顾 academic writing 与 citation suggestion，适合作为“综述写作 + 引用建议”参考对象。 | [ScholarCopilot 项目页](https://tiger-ai-lab.github.io/ScholarCopilot/) |

| 资源 | 类型 | 简述 | 链接 |

## 3. 这类资源最适合产出哪些文稿

| 文稿类型 | 更适合优先参考的资源 |
| --- | --- |
| rapid review | `Consensus Pro Analysis`、`Elicit Reports`、`Open Research` |
| literature review / related work | `Elicit`、`STORM`、`GPT Researcher`、`RTI` |
| systematic review | `Elicit Systematic Reviews` |
| technical report / evidence brief | `Consensus`、`Elicit Reports`、`Open Research` |
| citation-grounded draft | `ScholarCopilot`、`PaperQA2`、`Open Research` |

## 4. 一条比较稳的证据综合链路

1. 用 `Elicit / Consensus / OpenAlex / OpenReview` 做 broad search 与 research question 澄清。
2. 用 `RTI / GPT Researcher / STORM / PaperQA2` 做重点论文池整理与 evidence extraction。
3. 用 `Elicit Reports / Consensus Pro Analysis / ScholarCopilot` 形成首版报告或综述骨架。
4. 用 [学术写作与审稿](./学术写作与审稿.md) 里的写作与 rebuttal 工具完成后续成稿与修订。

## 5. 适合优先关注的几个评估点

| 评估点 | 说明 |
| --- | --- |
| 检索覆盖度 | 是否支持多轮检索、筛选与较大规模论文池。 |
| 引用可回查 | 是否能回到具体论文、片段或 supporting quotes。 |
| 筛选透明度 | 是否保留 screening criteria、排除理由与中间步骤。 |
| 报告可编辑性 | 是否允许修改 prompt、criteria、paper set 与输出结构。 |
| 结果导出 | 是否支持导出表格、报告、引用或过程记录。 |
