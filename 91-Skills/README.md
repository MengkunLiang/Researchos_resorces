# Skills（类型附录）

本目录保留为查全型附录，适合系统补 `Skills` 生态、跨宿主技能库、科研专题 skill bundles，以及不同研究阶段的高频技能组合。

## 这一层适合什么时候进入

1. 需要系统扫公开 skills 生态，而不是只看单个任务页里的少量条目。
2. 需要补 `Claude Code / Codex / Copilot / OpenHands / Hugging Face` 等不同宿主的技能实现。
3. 需要把“选题、精读、实验、写作”阶段中的短条目扩成更完整的技能参考页。
