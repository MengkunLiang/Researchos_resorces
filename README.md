## 组织原则

- 顶层主线优先围绕科研任务展开，便于直接查询和使用。
- `07-09` 负责横向治理、方向专题与任务使用导航。
- `91-97` 保留为查全型附录，适合系统扫库、补库和后续扩展。
- 同一外部资源的较完整说明尽量只保留在一个主归属页，其余页面以短说明和内链引用为主。
- 中文说明优先，收录对象以中英文公开资源并重，不限制资源原始语言。
