## 当前组织原则

- 顶层目录优先围绕科研任务展开，便于直接查询和使用。
- `91-97` 目录保留为查全型附录，适合系统扫库、补库和后续扩展。
- 同一外部资源的详细说明尽量只保留在一个主归属页，其余页面用短说明和内链引用。
- 外部公开仓库优先采用中文摘要、条目化整理和原链接回指，不做整页镜像式堆叠。
