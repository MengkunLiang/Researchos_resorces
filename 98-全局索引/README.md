# 全局索引

本目录用于横向查找、扫库和快速补入口。更适合在以下场景进入：

1. 需要快速找到某一类资源的仓库内主入口。
2. 需要继续补公开合集、优秀索引页或外部社区资源。
3. 需要从任务视角切回横向视角做查全与比较。

## 与其他层的关系

- 如果当前重点是“这一阶段该怎么做”，优先进入 [09-按科研任务导航](../09-按科研任务导航/README.md)。
- 如果当前重点是“这一阶段最该补哪几个方向专题”，优先进入 [任务专题对照索引](./任务专题对照索引.md)。
- 如果当前重点是“这类问题通常对应哪些正文、附录和专题”，优先进入 [00-总览/任务与资源对照表](../00-总览/任务与资源对照表.md)。
- 如果当前重点是“这个方向有哪些资源”，优先进入 [方向专题索引](./方向专题索引.md) 或 [08-方向专题](../08-方向专题/README.md)。
- 如果当前重点是“这类资源系统有哪些”，优先进入 [90-按类型参考](../90-按类型参考/README.md)。
- 如果当前重点是“这一步还缺哪些 Skills / MCP”，优先进入 [91-Skills](../91-Skills/README.md) 与 [92-MCP](../92-MCP/README.md)。

## 常见使用方式

1. 从 [00-总览/任务与资源对照表](../00-总览/任务与资源对照表.md) 找到当前问题所在层级。
2. 如果你知道自己在哪一步，先看 [任务专题对照索引](./任务专题对照索引.md)。
3. 如果你知道自己是什么方向，先看 [方向专题索引](./方向专题索引.md)。
4. 再进入本目录对应类型索引页，扫一遍仓库内主入口与高频公开资源。
5. 最后回到 `91-97` 附录或 `01-09` 主模块看更完整的正文说明。
