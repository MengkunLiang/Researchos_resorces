## 五、常见使用顺序

1. 先用 `OpenReview MCP / OpenAlex MCP / Paper Search MCP` 建立相关工作和评审语境。
2. 再用 `paper-glance-skill / literature-review / scientific-critical-thinking` 对核心论文做统一拆解。
3. 用 `llm4novelty / NovBench + ReviewAdvisor + NLPeer + PeerRead` 补 novelty 与 reviewer-style critique。
4. 若需要 claim 级证据核验，再接 `PeerQA / SciDQA / SciFact / SciFact-Open / scite`。
5. 若需要 systematic review 或 evidence synthesis，再接 `ASReview / SYNERGY / Elicit / Consensus`。
6. 最后把结论沉到 [03-知识管理与证据沉淀](../03-知识管理与证据沉淀/README.md) 或 [06-写作、审稿与投稿](../06-写作、审稿与投稿/README.md)。
