## 五、常见使用顺序

1. 先用 `PaperQA2 / Zotero / citation-management` 固定 claim 和证据链。
2. 再用 `OpenDraft / ScholarCopilot / academic-paper` 形成初稿与段落骨架。
3. 用 `peer-review / academic-paper-reviewer / AI Peer Review / Open Research` 做 critique 和 response points。
4. 若采用 programmable authoring，再接 `Manubot / MyST / Curvenote / Quarto / Typst`。
5. 进入投稿末端时，再接 [模板、Rebuttal与投稿系统](./模板、Rebuttal与投稿系统.md)。
