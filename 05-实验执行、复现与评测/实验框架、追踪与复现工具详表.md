## 六、常见使用顺序

1. 先用 `uv / micromamba / conda-lock` 固定环境，再决定是否需要 `Docker / Apptainer`。
2. 用 `Hydra + Optuna + MLflow / Aim` 管理配置、trial 和结果。
3. 如果数据与模型需要版本化，再接 `DVC` 或 `DataLad`。
4. notebook 主导的研究，优先接 `Jupyter MCP + Papermill / marimo + Quarto`。
5. 多步骤流水线优先接 `Prefect / Metaflow / Kedro / Snakemake / Nextflow / ZenML`。
6. 对公开可复现和长期保存要求较高时，再接 `ReproZip / Whole Tale / REANA`。
