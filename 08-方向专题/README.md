# 方向专题

这一模块用于维护跨模块的方向型专题页。主导航仍以科研子问题为主，方向专题负责把某一研究线上的 `Skills`、`MCP`、agent、benchmark、公共数据源、外部合集和代表性开源框架集中到同一入口，便于按研究方向持续补库。当前优先覆盖 `AI/ML`、`data mining`、`computational design science` 与开放创新相关主题；`scientific discovery` 相关内容保留为补充支线。

## 这一步通常要回答的问题

1. AI/ML、data mining、computational design science、开放创新与社会技术研究分别有哪些最值得优先接入的公共资源。
2. 每一类方向更适合搭配哪些 `Skills`、`MCP`、实验系统、评测框架与数据入口。
3. 哪些 benchmark、开放数据和公开合集最适合继续深挖，而不是停留在零散链接收集。
4. 方向专题与任务主线之间应当怎样对应，例如 novelty 判断、实验执行、写作投稿和安全治理分别如何回链。
